# Formulario REST configurable

Aplicación web MVP para enviar solicitudes mediante una API REST configurable.

## Requisitos

- Node.js 20+
- npm

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Compilación

```bash
npm run build
```

## Pruebas

```bash
npm test
```

## Rutas

- `/` — Formulario principal
- `/admin` — Configuración (sin enlace visible desde la página principal)
